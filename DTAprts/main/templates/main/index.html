DTAprts

Добро пожаловать в DTAprts — ваш помощник в поиске идеальной квартиры в Москве!

Цели проекта

Основная цель проекта — разработать удобный и функциональный инструмент, который поможет пользователям: Быстро находить актуальные объявления об аренде квартир. Фильтровать предложения по ключевым параметрам (цена, площадь, комнаты, комиссия). Анализировать рынок аренды через визуализацию данных (графики цен, средней стоимости по районам и зависимости цены от площади).

О проекте

DTAprts — это веб-приложение, созданное для упрощения поиска жилья в аренду в Москве. Проект вдохновлён личным опытом автора, который столкнулся с трудностями при поиске подходящей квартиры в столице.

Практическая направленность

Проект направлен на создание приложения, облегчающего поиск квартир в Москве в настоящее время. Идея возникла из личной необходимости: я сам сталкивался с проблемой поиска подходящего жилья для аренды в Москве, где множество объявлений, сложные фильтры и отсутствие удобного анализа данных затрудняют процесс. DTAprts решает эту задачу, предоставляя простой интерфейс и аналитические инструменты.

Результат проекта

Итогом работы стало веб-приложение с:

  • Интерактивными фильтрами для поиска квартир по цене, количеству комнат, площади и комиссии.
  • Динамическими графиками, отображающими распределение цен, среднюю стоимость по районам и зависимость цены от площади.
  • Списком отфильтрованных объявлений с прямыми ссылками на оригинальные предложения.
  • Новостной лентой с инструментом для управления новостным контентом на сайте. Оно предоставляет простой интерфейс для создания, редактирования и отображения новостей, интегрируясь Django-приложением news. Будь то анонсы, обновления или статьи, news помогает организовать контент и донести его до пользователей.
  • Удобным и современным дизайном, адаптированным для пользователей.

Краткий план реализации

Проект был реализован в несколько этапов:

  1. Сбор данных: Парсинг объявлений с сайта ЦИАН и сохранение их в формате JSON.
  2. Создание базы данных: Настройка модели `Apartment` в Django для хранения данных об объявлениях.
  3. Разработка backend: Реализация логики фильтрации и обработки данных с использованием Python и Pandas.
  4. Визуализация: Интеграция библиотеки Plotly для построения интерактивных графиков.
  5. Frontend: Создание интерфейса с фильтрами, списком объявлений и динамическим отображением с помощью HTML, CSS и JavaScript.
  6. Тестирование и доработка: Проверка функциональности и улучшение пользовательского опыта.

Используемые технологии

Проект построен с использованием следующих фреймворков, языков и инструментов:

  • Django: Основной фреймворк для backend-разработки и управления базой данных.
  • Python: Язык программирования для логики приложения и обработки данных.
  • PostgreSQL/SQLite: База данных для хранения объявлений и связанных с ними данных.
  • Selenium: Библиотека для автоматизации браузера и парсинга динамических страниц.
  • ChromeDriver: Инструмент для управления Chrome через Selenium.
  • Regular Expressions (re): Для извлечения чисел (комнат, площади, комиссии) из текста.
  • JSON: Формат хранения данных для удобного экспорта после парсинга.
  • Pandas: Библиотека для анализа и фильтрации данных.
  • Plotly: Инструмент для создания интерактивных графиков.
  • HTML: Структура страниц приложения.
  • CSS: Стилизация интерфейса (включая кастомный дизайн с градиентами и эффектами).
  • JavaScript: Динамическое отображение списка объявлений (функция "Показать ещё"), настройка фильтров.
  • Docker: Контейнеризация приложения для упрощения развертывания.
  • Git и GitHub: Контроль версий и хранение кода.
  • Yandex Cloud: Облачная платформа, использованная для размещения сайта. В рамках проекта была создана виртуальная машина (VM) на базе Yandex Compute Cloud, настроен публичный сервер с выделенным IP-адресом, установлен веб-сервер для запуска Django-приложения. Это обеспечило доступность сайта через интернет и стабильную работу в продакшене.
